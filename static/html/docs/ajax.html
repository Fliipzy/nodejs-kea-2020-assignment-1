AJAX

Asynchronous JavaScript And XML (AJAX), is a really handy js technology. With it, we can update web pages without having to reload the page.
We can also request and receive data from webservers after the page has fully loaded.

It may be hard to appreciate if one doesn't understand the alternative. Imagine having to reload every webpage ever, once you click a button that sends data to the web-server. Not being able to send or recieve data after the page has loaded.
Almost every popular website today uses AJAX to make things work, behind the scenes.

It's name is quite misleading in that we don't have to use XML to transport data, it's equally common to use JSON format or plain text.

Try it yourself

I've set up an example that uses vanilla (No 3rd party libs) AJAX to asynchronously (not having to reload page) fetch jokes from a public REST API called icanhazdadjoke.

If you click the button below, an asynchronous GET request will be made to the API and the result will be displayed below.





Heres the javascript code that makes this possibly: 


let jokeText = document.getElementById('jokeText');

//AJAX function
function getRandomJoke() {
    var xhttp = new XMLHttpRequest();
    xhttp.onreadystatechange = function() {
        if (this.readyState == 4 && this.status == 200) {
            let json = JSON.parse(this.responseText);
            jokeText.innerText = json['joke'];
        }
    };
    xhttp.open("GET", "https://icanhazdadjoke.com/", true);
    xhttp.setRequestHeader("Accept", "application/json");
    xhttp.send();
}
