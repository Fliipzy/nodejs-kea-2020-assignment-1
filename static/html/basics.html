Node.JS basics

What is it?

Node.JS is a runtime enviroment for running JavaScript code outside the browser, which you couldn't do before.
It runs on the very popular JavaScript V8 engine created by the Google Chrome people.

What can I do with it?

You can do alot with Node.JS. Wanna build easy scalable back-end web applications,
maybe desktop applications or use it's awesome package manager NPM?
These are just some of the things you can do with node.

JavaScript basics

Variables

JavaScript has loosely-typed variables. It means variables aren't declared with a particular data type, like in Java or C#. Instead you can declare variables using the keywords: var, let or const. In fact you can declare a variable by providing only it's name and value, that is however not recommended because it can accidently overwrite an existing global variable.

Variables are used for storing data values, like almost any other language.
Heres an example, where we declare two variables using the var keyword: 


var x = 10;
var y = 'Hello, World';

In the above example, we declared 2 variables:
We assigned the numeric value 10 to x.
We assigned a string value to y.

Difference between var and let?

The keywords var and let are used for the exact same thing, declaring variables.
The difference between the two are what scope the variable will be in:

With let, the variable will only exist in the scope it was declared in. In other words, it's block scoped


for(let i = 0; i < 10; i++){
    console.log(i); 
    //prints 0,1,2..,9
}
console.log(i);
//i will not exist here and the program will crash

With var, the variable is accessible from the whole program. 


console.log(i);
//prints undefined

for(var i = 0; i < 10; i++){
    console.log(i); 
    //prints 0,1,2..,9
}
console.log(i);
//prints 10

There's a general consensus in the JS community, to never use var unless you have to.


Functions

Functions in JavaScript is like any other programming language, just a block of code that should serve to do a specific thing.

Heres an example of a function that returns the sum of it's two parameters: 


function addNumbers(n1, n2) {
    return n1 + n2;
}

Functions are executed when it's invoked, like this: 


addNumbers(5, 3);
